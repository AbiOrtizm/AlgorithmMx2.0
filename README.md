# Portafolio Personal Next.js

Portafolio personal construido con Next.js y preparado para desplegar en Vercel.

## Comenzar

1. Instalar dependencias:

```bash
npm install
```

2. Ejecutar en desarrollo:

```bash
npm run dev
```

3. Generar producción:

```bash
npm run build
npm start
```

## Despliegue en Vercel

- Conecta este proyecto a Vercel.
- Vercel detecta Next.js automáticamente.
- Si necesitas, usa `vercel.json` para la configuración de despliegue.
